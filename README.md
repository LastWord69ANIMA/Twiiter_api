上記エラーの原因として「twitterからxへの移行に伴う、apiの有料化やその後の無料化などの騒動」に起因するapiの仕様変更と存じています。
今後はxのapi仕様に従い、作り直そうと存じます。また、その際、.env等を上手く活用し、伏せ字･･･例えば、`CHANNEL_ACCESS_TOKEN = "****"`等をわざわざ載せることなくデプロイしたいと思っています。
